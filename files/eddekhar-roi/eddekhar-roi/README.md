# Eddekhar ROI Calculator | حاسبة العائد على الاستثمار

Interactive ROI calculator for the Eddekhar employee savings platform.

## Features
- Arabic/English bilingual with RTL/LTR toggle
- Real-time ROI calculations with interactive sliders
- Responsive design (desktop + mobile)
- Charts: bar, pie, area with tooltips

## Deploy
```bash
npm install
npm run build
npm start
```

## Tech
- React 18 + Vite
- Recharts
- Deployed on Railway
